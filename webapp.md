# Web Application Programming

935 304 การเขียนโปรแกรมประยุกต์บนเว็บ

### คำอธิบายรายวิชา

เว็บเซิร์ฟเวอร์ เว็บบราวเซอร์ สคริปฝั่งไคลเอนท์ สคริปฝั่งเซิร์ฟเวอร์ เครื่องให้บริการฐานข้อมูล การเขียนโปรแกรมบนเว็บสำหรับเนื้อหาเชิงโต้ตอบ การเขียนโปรแกรมบนเว็บสำหรับเนื้อหาแบบพลวัต การเขียนโปรแกรมสำหรับฐานข้อมูลบนเว็บ การเขียนโปรแกรมบนเว็บให้มีความมั่นคง การใช้เฟรมเวิร์กสำหรับเขียนโปรแกรมบนเว็บ
