# การเขียนโปรแกรมด้วยภาษา TypeScript

TypeScript คือภาษาโปรแกรมมิ่งที่พัฒนาโดย Microsoft เป็น Superset ของ JavaScript หมายความว่าโค้ด JavaScript ทุกตัวสามารถทำงานบน TypeScript ได้ แต่ TypeScript เพิ่มความสามารถด้าน **Static Type Checking** เข้ามา ช่วยให้ตรวจพบข้อผิดพลาดได้ตั้งแต่ขั้นตอนการเขียนโค้ด ก่อนที่จะนำไปรันจริง
