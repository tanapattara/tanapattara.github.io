# การเขียนโปรแกรมบนอุปกรณ์เคลื่อนที่ด้วย React Native

## React Native คืออะไร

React Native คือ Framework สำหรับพัฒนาแอปพลิเคชันบนอุปกรณ์เคลื่อนที่ (Mobile Application) ที่สามารถทำงานได้ทั้งบน **iOS** และ **Android** จากโค้ดชุดเดียวกัน พัฒนาโดย Meta (Facebook) และเผยแพร่เป็น Open Source ในปี ค.ศ. 2015 โดยใช้ภาษา **JavaScript** ร่วมกับไลบรารี **React** เป็นพื้นฐาน

ความแตกต่างจาก WebView-based framework อื่น ๆ คือ React Native แปลง Component ที่เขียนด้วย JavaScript ให้กลายเป็น Native UI Component ของแต่ละแพลตฟอร์ม ทำให้ได้ประสิทธิภาพและหน้าตาที่ใกล้เคียงกับแอปพลิเคชันที่เขียนด้วยภาษา Native (Swift / Kotlin) อย่างแท้จริง

## ทำไมต้องเรียน React Native

- **Cross-platform** — เขียนโค้ดครั้งเดียว รันได้ทั้ง iOS และ Android
- **ใช้ TypeScript** — ภาษาที่ type-safe ช่วยลด bugs และเพิ่มประสิทธิภาพการพัฒนา
- **Hot Reload** — เห็นผลการเปลี่ยนแปลงทันทีโดยไม่ต้องรีสตาร์ทแอป
- **Ecosystem ขนาดใหญ่** — มี Library และ Community พร้อมรองรับการพัฒนา
- **ใช้งานจริงในองค์กรชั้นนำ** — เช่น Meta, Microsoft, Shopify, Discord

## เนื้อหาในส่วนนี้

หัวข้อต่าง ๆ ในส่วนนี้ถูกแบ่งออกเป็นบทเรียนย่อย เหมาะสำหรับนักศึกษาที่เริ่มต้นจนถึงระดับกลาง โดยเรียงลำดับจากพื้นฐานไปสู่การประยุกต์ใช้งานจริง ดังนี้

| บท | หัวข้อ | สิ่งที่จะได้เรียนรู้ |
| --- | --- | --- |
| 1 | การเริ่มต้นและติดตั้ง | ติดตั้ง Environment, สร้างโปรเจกต์แรกด้วย TypeScript |
| 2 | Core Components | View, Text, Image, TextInput, Button |
| 3 | การจัดรูปแบบและ Styling | StyleSheet, Flexbox, Responsive Design |
| 4 | Navigation | React Navigation, Stack, Tab, Drawer |
| 5 | State Management | useState, useReducer, Context API |
| 6 | API และ Networking | fetch, axios, REST API |
| 7 | Local Storage | AsyncStorage, SecureStore |
| 8 | Workshop | แอปพลิเคชันตัวอย่างแบบครบวงจร |
