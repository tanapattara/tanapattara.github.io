# การเขียนโปรแกรมด้วยภาษา C# 

## ความเป็นมาและลักษณะของภาษา C#

ภาษา C# (อ่านว่า "ซีชาร์ป") ได้รับการพัฒนาโดย Microsoft ภายใต้การนำของ Anders Hejlsberg และเปิดตัวครั้งแรกในปี ค.ศ. 2000 พร้อมกับแพลตฟอร์ม .NET Framework ภาษา C# ถูกออกแบบให้เป็นภาษาเชิงวัตถุ (Object-Oriented Programming) ที่มีความปลอดภัยของประเภทข้อมูล (Type-Safe) และรองรับการพัฒนาโปรแกรมได้หลากหลายรูปแบบ ทั้งแอปพลิเคชันบนเดสก์ท็อป เว็บแอปพลิเคชัน บริการ API และแอปพลิเคชันบนอุปกรณ์เคลื่อนที่

ภาษา C# มีลักษณะเด่นหลายประการ ได้แก่ การเป็นภาษาที่ต้องกำหนดชนิดข้อมูลอย่างชัดเจน (Strongly Typed) ซึ่งช่วยลดข้อผิดพลาดในระหว่างการทำงานของโปรแกรม มีระบบกำจัดหน่วยความจำอัตโนมัติ (Garbage Collection) และรองรับการเขียนโปรแกรมแบบ Asynchronous อย่างเต็มรูปแบบ

## โครงสร้างพื้นฐานของโปรแกรม C#

โปรแกรมภาษา C# ทุกโปรแกรมมีโครงสร้างพื้นฐานที่ต้องประกอบด้วยส่วนสำคัญหลายส่วน ได้แก่ การประกาศ Namespace การกำหนด Class และ Method หลักที่ชื่อว่า `Main` ซึ่งเป็นจุดเริ่มต้นการทำงานของโปรแกรม

```csharp
using System; // นำเข้า Namespace ของระบบ

namespace HelloWorldApp // กำหนด Namespace ของโปรแกรม
{
    class Program // กำหนด Class หลัก
    {
        static void Main(string[] args) // Method หลักที่เป็นจุดเริ่มต้น
        {
            Console.WriteLine("สวัสดีโลก! Hello, World!");
            Console.ReadLine(); // รอรับ Input เพื่อไม่ให้หน้าต่างปิด
        }
    }
}
```

**คำอธิบายโค้ด:**
`using System` คือการนำเข้าไลบรารีพื้นฐานของ .NET ที่ให้เราใช้คำสั่งพื้นฐานได้ `namespace` ใช้สำหรับจัดกลุ่มของ Class เพื่อป้องกันการชนกันของชื่อ และ `Main` คือ Method ที่โปรแกรมจะเริ่มทำงานจากจุดนี้เสมอ
