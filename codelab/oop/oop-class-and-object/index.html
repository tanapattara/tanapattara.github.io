การทดลอง

การทดลองนี้เป็นการสร้างคลาส และทำความเข้าใจองค์ประกอบของคลาสในภาษาจาวา การสร้าง Attribute และ Method ของคลาส โดยทำการสร้างคลาสนักศึกษาเพื่อทำการแทนตัวนักศึกษา และทำการอ่านข้อมูลนักศึกษาจากไฟล์ csv ที่เก็บข้อมูลนักศึกษา และแสดงรายชื่อนักศึกษาทั้งหมดในคลาสรายวิชา ผ่านทาง console

วัตถุประสงค์การทดลอง

  • ทำความเข้าใจการสร้างและองค์ประกอบของคลาสและวัตถุ
  • การสร้างคลาสและวัตถุ
  • การเข้าถึงและเรียกใช้งานวัตถุ
  • การอ่านไฟล์และการสร้างวัตถุโดยใช้ข้อมูลจากไฟล์ csv

ต้องมีอะไรบ้างก่อนทำ

  • ความรู้ความเข้าใจการเขียนโปรแกรมขั้้นพื้นฐาน
  • ความรู้ความเข้าใจภาษาจาวา
  • IntelliJ IDEA หรือ IDE อื่นๆ ในการพัฒนาโปรแกรมจาวา

เริ่มต้นสร้างโปรเจคขึ้นมาใหม่ ตามเครื่องมือที่นักศึกษาได้ใช้ ในเลปนี้เป็นการใช้เครื่องมือ IntelliJ IDEA ในการสร้างโปรเจค โดยกำหนดให้สร้างโปรเจคชื่อ StudentOOP

เลือกรูปแบบ Command Line App

ตั้งชื่อโปรเจค และแพคเกจ ให้กับโปรเจคของเรา

IntelliJ IDEA จะสร้างไฟล์ขึ้นมาตามรูปแบบที่เราได้เลือกนั่นคือ Command Line โดยสร้างไฟล์ Main.java พร้อมเมธอท main() ให้เรา หากเราไม่ได้เลือกรูปแบบไว้ เราสามารถสร้างไฟล์ดังกล่าว พร้อมกับใส่ชุดคำสั่งด้านล่างลงไป

Main.java

package th.ac.kku.cis.lab;

public class Main {

    public static void main(String[] args) {
        // write your code here
    }
}

Note: ชื่อ package จะแตกต่างกันไปในแต่ละเครื่อง และเราสามารถแก้ไขชื่อได้ตั้งแต่ตอนสร้างโปรเจค หากในตอนเริ่มสร้างไม่ได้กำหนดชื่อเหมือนกับในเอกสาร ก็จะปรากฎชื่อที่แตกต่างออกไป แต่โปรแกรมยังทำงานได้

ภาพรวมของระบบ

ก่อนจะทำการเขียนโปรแกรม เรามาทราบถึงความต้องการของระบบก่อน ระบบของเราเป็นโปรแกรมที่ใช้เก็บข้อมูลนักศึกษาที่เข้าเรียนในรายวิชา การเขียนโปรแกรมเชิงวัตถุ โดยข้อมูลนักศึกษาประกอบไปด้วย ชื่อ นามสกุล รหัสนักศึกษา อีเมล

จากความต้องการของระบบ นักศึกษาที่เรียนภายในรายวิชา มีข้อมูลของตัวนักศึกษา จากรายละเอียดนี้ เราจะนำมาประกอบเป็นคลาส

สร้างคลาสนักศึกษา

สร้างไฟล์ใหม่ เลือกเป็น New Java class ตั้งชื่อว่า Student ทำให้เราได้ไฟล์ Student.java ในการเขียนโปรแกรม เรานิยมตั้งชื่อไฟล์ให้ตรงกับคลาส ถึงแม้ว่าบางภาษาจะอนุญาติให้มีการสร้างคลาสย่อย (Subclass) ในไฟล์เดียวกันได้ แต่เราสร้างคลาสไว้เพียงหนึ่งคลาส ต่อไฟล์หนึ่งไฟล์

หลังจากสร้างไฟล์แล้ว เราจะได้คลาส Student เปล่าๆ มาหนึ่งคลาส ดังนี้

Student.java

package th.ac.kku.cis.lab;

public class Student{

}

ทำการเพิ่มรายละเอียดของคลาส ตามที่ความต้องการของระบบระบุไว้ นั่นก็คือ ชื่อ นามสกุล รหัสนักศึกษา และอีเมล จากข้อมูลคุณลักษณะนี้ เราจะสร้างเป็นตัวแปรของคลาส หรือแอททริบิวท์ของคลาส (class' attribute)

Student.java

package th.ac.kku.cis.lab;

public class Student{
     String firstname;
     String lastname;
     String student_id;
     String email;
}

ตอนนี้คลาสนักศึกษาของเราพร้อมในระดับหนึ่ง ในการนำไปสร้างเป็นวัตถุแล้ว

หลังจากเราได้สร้างคลาสนักศึกษา ที่เป็นเสมือแม่แบบ ที่ระบุโครงสร้างของวัตถุ เราจะนำคลาสที่เราสร้างขึ้น มาสร้างเป็นวัตถุ เพื่อนำมาใช้งาน แทนนักศึกษาแต่ละคน

ที่ไฟล์ Main.java เพิ่มคำสั่งสร้างวัตถุต่อไปนี้

Main.java

package th.ac.kku.cis.lab;

public class Main {

    public static void main(String[] args) {
         //Create object
         Student student = new Student();
    }
}

คำสั่งข้านบนเป็นการสร้างวัตถุขึ้นมาจากคลาส Student โดยตั้งชื่อวัตถุนั้นว่า student

Note: ลักษณะการสร้างวัตถุ จะคล้ายกับการสร้างตัวแปร โดยมีการกำหนดชนิดของข้อมูลก่อน ในคำสั่งช้างบนเรากำหนดชนิดข้อมูลเป็นคลาส Student ที่เราสร้างขึ้น

Note: คลาสที่เราใช้งานในการสร้างวัตถุคือ Student ขึ้นต้นด้วยตัวใหญ่ แต่วัตถุเราสร้างขึ้นมาชื่อ student ใช้ตัวแรกเป็นตัวเล็ก ตามกฏการตั้งชื่อตัวแปรในการเขียนโปรแกรจะมองเป็นคนละตัว แตกต่างกัน

วัดถุที่เราสร้างขึ้น เราจะใช้แทนข้อมูลนักศึกษา โดยเราจะเก็บข้อมูลลงแอททริบิวต์ของคลาสผ่านวัตถุ โดยใช้คำสั่ง

Main.java

        //Create object
        Student student = new Student();

        //Use object student
        student.firstname = "Mickey";
        student.lastname = "Mouse";
        student.student_id = "001";
        student.email = "mickey@disney.com";

หลังจากใส่ข้อมูลนักศึกษาลงไปในวัตถุแล้ว วัตถุจะเก็บข้อมูลเหล่านั้นไว้ภายใน โดยโครงสร้างของวัตถุนั้น ก็ถูกกำหนดจากคลาสที่วัตถุถูกสร้างขึ้น นั่นก็คือคลาส Student

วัตถุ student นี้ก็จะเก็บข้อมูลของนักศึกษาที่ชื่อ Mickey ไว้ แล้วถ้าเกิดอยากเก็บข้อมูลนักศึกษาคนอื่นเพิ่มหละ ? เราก็ต้องสร้างวัตถุนักศึกษาขึ้นมาใหม่ เพื่อเก็บข้อมูลนักศึกษาอีกคน เพราะเราออกแบบให้คลาส Student แทนข้อมูลนักศึกษาเพียงคนเดียว

สร้างวัตถุเพิ่ม และใส่ข้อมูลนักศึกษาคนที่สองลงไป ต่อจากคำสั่งเดิม

Main.java

        //Create object
        Student student1 = new Student();

        //Use object student
        student1.firstname = "Minnie";
        student1.lastname = "Mouse";
        student1.student_id = "002";
        student1.email = "minnie@disney.com";

ตอนนี้เรามีนักศึกษาอยู่สองคน นั่นคือวัตถุ student และ student1 โดยแต่ละวัตถุมีข้อมูลของนักศึกษาแต่ละคนแยกจากกัน

เรียกใช้วัตถุ

ลองดึงข้อมูลจากวัตถุ student และ student1 ให้แสดงชื่อและนามสกุลออกทางคอนโซล โดยใช้คำสั่ง

Main.java

System.out.println(String.format("Student name %s %s",student.firstname, student.lastname));
System.out.println(String.format("Student name %s %s",student1.firstname, student1.lastname));

ทดสอบและแสดงผลการทำงาน

จากความต้องการของระบบ เป็นระบบเก็บข้อมูลนักศึกษาที่เข้าเรียนในรายวิชา ตอนนี้เราได้คลาสนักศึกษา สำหรับใช้ในการสร้างวัตถุนักศึกษาแล้ว ยังขาดรายรายวิชาที่เอาไว้เก็บข้อมูลนักศึกษาภายในชั้นเรียน

สร้างคลาสรายวิชา

สร้างคลาสใหม่ โดยกำหนดชื่อไฟล์ว่า OOPClass ทำให้เราได้ชื่อคลาสว่า OOPClass ภายในไฟล์ OOPClass.java ตามลำดับ

จากรายละเอียดของความต้องการ คลาสนี้จะบรรจุข้อมูลนักศึกษาที่เข้าเรียนไว้ ทำให้ต้องมีการกำหนดแอททริบิ้วท์ของคลาส ไว้เก็บข้อมูลนักศึกษา โดยตั้งชื่อว่า students

OOPClass.java

package th.ac.kku.cis.lab;

public class OOPClass {
    Student students;
}

จากแนวคิดของการเขีนโปรแกรมเชิงวัตถุและการสร้างวัตถุ Student ก่อนหน้า จะเห็นได้ว่าวัตถุ Student แทนนักศึกษา 1 คน แต่รายวิชาที่เราสร้างขึ้นนี้ไม่ได้ระบุจำนวนนักศึกษาว่านักศึกษามีจำนวนเท่าไหร่ แต่เรามั่นใจได้ว่า 1 รายวิชาต้องมีนักศึกษามากกว่า 1 คนแน่นอน ทำให้เราต้องเปลี่ยนแปลงชนิดข้อมูลเพื่อให้สอดคล้องกับความต้องการนี้

หากเราต้องการเก็บข้อมูลวัตถุนักศึกษา ที่มากกว่า 1 คน โดยชนิดข้อมูลยังคงเป็นนักศึกษาอยู่เหมือนๆ กัน เราสามารถใช้ Array ในการเก็บข้อมูลได้ แต่ array มีข้อจำกัดในการสร้างและการเพิ่มขนาด ดังนั้นเพื่อความสามารถในการเพิ่มวัตถุนักศึกษา เราสามารถใช้ ArrayList แทน

OOPClass.java

package th.ac.kku.cis.lab;

import java.util.ArrayList;

public class OOPClass {
    ArrayList<Student> students;
}

ในคลาสรายวิชาที่เก็บข้อมูลนักศึกษาที่ลงทะเบียนไว้ ดังนั้นเราต้องมีการสร้างวัตถุนักศึกษา และทำการเก็บวัตถุนั้นไว้ในคลาสรายวิชา

ย้ายคำสั่งสร้างวัตถุนักศึกษาในไฟล์ Main.java มาไว้ที่ OOPClass.java โดยทำการสร้างเมธอทขึ้นมาใหม่ชื่อว่า createStudent()

OOPClass.java

public class OOPClass {
    ArrayList<Student> students;

    public void createStudent(){
        Student student = new Student();
        student.firstname = "Mickey";
        student.lastname = "Mouse";
        student.student_id = "001";
        student.email = "mickey@disney.com";

        Student student1 = new Student();
        student1.firstname = "Minnie";
        student1.lastname = "Mouse";
        student1.student_id = "002";
        student1.email = "minnie@disney.com";
    }
}

ทำการบันทึกวัตถุนักศึกษาลง students ผ่านคำสั่ง students.add()

OOPClass.java

public class OOPClass {
    ArrayList<Student> students;

    public void createStudent(){
        Student student = new Student();
        student.firstname = "Mickey";
        student.lastname = "Mouse";
        student.student_id = "001";
        student.email = "mickey@disney.com";

        Student student1 = new Student();
        student1.firstname = "Minnie";
        student1.lastname = "Mouse";
        student1.student_id = "002";
        student1.email = "minnie@disney.com";

        students.add(student);
        students.add(student1);
    }
}

ตอนนี้ไฟล์คลาสรายวิชาของเรา มีข้อมูลนักศึกษาสองคนที่ลงทะเบียนรายวิชาเก็บเอาไว้ในคลาสเรียบร้อย ทดสอบเรียกข้อมูลนักศึกษาที่ลงทะเบียน ผ่านคลาสรายวิชา ที่ไฟล์ Main.java ดังนี้

Main.java

package th.ac.kku.cis.lab;

import java.util.ArrayList;

public class Main {

    public static void main(String[] args) {
        //Create object
        OOPClass oopClass = new OOPClass();
        //Create student data
        oopClass.createStudent();
        //get student data from class
        ArrayList<Student> students = oopClass.students;
        //display data
        System.out.println(String.format("Student name %s %s", students.get(0).firstname, students.get(0).lastname));
        System.out.println(String.format("Student name %s %s", students.get(1).firstname, students.get(1).lastname));
    }
}

เมื่อทดสอบการทำงานจะได้ผลลัพท์ แสดงข้อผิดพลาดดังนี้

Error: Exception in thread "main" java.lang.NullPointerException: Cannot invoke "java.util.ArrayList.add(Object)" because "this.students" is null

ข้อผิดพลาดเกิดจากตัวแปร students ที่ไฟล์ OOPClass.java มีค่าเป็น null ทำให้ไม่สามารถใส่ข้อมูลนักศึกษาลงไปได้ เนื่องจาก students เป็น ArrayList ต้องมีการประกาศค่าให้กับ students ให้เรียบร้อยซะก่อน

สร้างคอนสตรัคเตอร์

ที่ OOPClass.java เพิ่มคอนสตรัคเตอร์และประกาศค่าเริ่มต้นให้ students

OOPClass.java

public class OOPClass {
    ArrayList<Student> students;
    public OOPClass(){
        students = new ArrayList<Student>();
    }
    //...
}

หากสังเกตุชุดคำสั่งให้ดีคอนสตรัคเตอร์ ก็คือเมธอทที่ชื่อเดียวกับคลาส นั่นคือ OOPClass() คอนสตรัคเตอร์นี้จะทำงานเมื่อมีการสร้างวัตถุของคลาสครั้งแรก นั่นก็คือ หากมีการสร้างคลาสรายวิชา โปรแกรมจะมีการเรียกใช้คอนสตรัคเตอร์ของคลาสทันทีเป็นอันดับแรก

จากกระบวนการทำงานของคอนสตรัคเตอร์ หากเราต้องการให้มีการสร้างรายชื่อนักศึกษาพร้อมๆกับการสร้างคลาสรายวิชา ให้เราเรียกใช้เมธอท createStudent() ที่คอนสตรัคเตอร์ของคลาสรายวิชาได้เลย

ปรับปรุงคอนสตรัคเตอร์ให้เรียกใช้เมธอท จะได้

OOPClass.java

package th.ac.kku.cis.lab;

import java.util.ArrayList;

public class OOPClass {
    ArrayList<Student> students;
    public OOPClass(){
        students = new ArrayList<Student>();
        createStudent();
    }
    public void createStudent(){
        Student student = new Student();
        student.firstname = "Mickey";
        student.lastname = "Mouse";
        student.student_id = "001";
        student.email = "mickey@disney.com";

        Student student1 = new Student();
        student1.firstname = "Minnie";
        student1.lastname = "Mouse";
        student1.student_id = "002";
        student1.email = "minnie@disney.com";

        students.add(student);
        students.add(student1);
    }
}

ปรับปรุงการเรียกข้อมูลนักศึกษา ที่ไฟล์ Main.java เนื่อจากเรามีการเพิ่มเมธอทสำหรับสร้างรายชื่อนักศึกษาไว้ในคอนสตรัคเตอร์ ของคลาสรายวิชาแล้ว ทำให้เราสามารถลบการเรียกใช้งานเมธอท createStudent() ได้เลย ทำให้ชุดคำสั่งของเราเป็นดังนี้

        ArrayList<Student> students = oopClass.students;
        //display data
        System.out.println(String.format("Student name %s %s", students.get(0).firstname, students.get(0).lastname));
        System.out.println(String.format("Student name %s %s", students.get(1).firstname, students.get(1).lastname));
    }
}

เราสามารถนำคำสั่งการทำซ้ำ foreach มาช่วยในการแสดงผลข้อมูลจาก ArrayList ได้เพื่อรองรับหากมีข้อมูลของนักศึกษาที่เพิ่มมากขึ้น

Main.java

//display data
for (Student student: students) {
        System.out.println(String.format("Student name %s %s", student.firstname, student.lastname));
}

ตอนนี้การได้วัตถุของรายวิชาที่มีข้อมูลรายชื่อนักศึกษาบรรจุอยู่ภายในแล้ว ต่อไปเราจะลองเพิ่มเมธอทให้กับคลาสรายวิชาเพื่อให้วัตถุนี้ทำงานได้สมบูรณ์มากยิ่งขึ้น

เพิ่มเมธอทแสดงจำนวนนักศึกษาทั้งหมด

บ่อยครั้งที่เราจะต้องการทราบจำนวนนักศึกษาภายในคลาสว่ามีจำนวนเท่าไหร่ เราจะเพิ่มเมธอทสำหรับแสดงจำนวนนักศึกษาทั้งหมดไว้ให้เรียกใช้งาน โดยกำหนดให้ส่งจำนวนนักศึกษากลับหลังจากเรียกใช่เมธอทนี้ เพื่อนำไปคำนวน หรือแสดงผลต่อไป

เพิ่มเมธอท getStudentCount() ที่คลาสรายวิชา และส่งจำนวนนักศึกษาที่มีอยู่ภายในคลาสกลับไป

OOPClass.java

public int getStudentCount(){
    return this.students.size();
}

จากเมธอทข้างบน เรากำหนดให้ข้อมูลที่ถูกส่งกลับ มีชนิดข้อมูลเป็น int และทำการส่งค่ากลับ (return) โดยใช้เมธอท size() จากตัวแปร students

หาเราต้องการเรียกใช้งาน ก็เรียกใช้ผ่านวัตถุของรายวิชา ดังนี้

int studentsSize = oopClass.getStudentCount();

จะเห็นได้ว่า เราเรียกเมธอทผ่านวัตถุของรายวิชา และนำตัวแปร์ studentsSize มารับข้อมูลจากเมธอทไว้ ทำให้เราทราบจำนวนนักศึกษาที่เรียนอยู่ในรายวิชานี้ทั้งหมด

ตอนนี้เราใช้การสร้างข้อมูลนักศึกษาขึ้นมาภายในคลาสรายวิชา ในความเป็นจริง เราต้องมีการนำเข้าข้อมูลนักศึกษาจากภายนอก ไม่ว่าจะเป็นฐานข้อมูล หรือไฟล์รายชื่อ ต่อไปนี้จะเป็นการอ่านข้อมูลจากไฟล์นามสกุล csv แล้วทำการบันทึกลงวัตถุนักศึกษา เพื่อเตรียมใช้งานในโปรแกรมต่อไป

สร้างเมธอทสำหรับอ่านข้อมูล

เพิ่มเมธอดใหม่สำหรับอ่านข้อมูลจาก CSV ที่ไฟล์ OOPClass.java ใช้คลาส BufferedReader ในการอ่านไฟล์โดยกำหนดตำแหน่งของไฟล์ที่ต้องการอ่านในเครื่อง และกำหนด charset สำหรับการอ่านข้อมูลตัวอักษร

BufferedReader br = Files.newBufferedReader(pathToFile, StandardCharsets.UTF_8)

เมื่อ BufferedReader เข้าถึงไฟล์ที่กำหนเราจะให้ อ่านข้อมูลมาทีละบรรทัด ( 1 บรรทัดก็คือข้อมูลของนักศึกษา 1 คน ในไฟล์ csv)

String line = br.readLine();

เราจะได้ตัวแปร line แทนข้อมูล 1 บรรทัดจากไฟล์ csv ดังนั้น บรรทัดแรกที่โปรแกรมจะอ่านได้ นั่นคือ

01,Luke,Skywalker,skywalker@rebelalliance.com

สังเกตุได้ว่า ข้อมูลแต่ละบรรทัดจะเรียงข้อมูลเป็น id, firstname, lastname, email โดยมีเครื่องหมาย "," เป็นตัวขั้น

เราจะทำการแยกข้อมูลเหล่านี้ออกจากกัน โดยใช้คำสั่ง และระบุเครื่องหมายในการแยกตัวอักษรออกจากกันนั่นคือเครื่องหมาย ","

String[] attributes = line.split(",");

ผลลัพท์ที่ได้เราจะได้คัวแปร attributes เป็น string array ซึ่งจะเก็บข้อมูลแยกกันตามอินเด็กของอาร์เรย์ ตามลักษณะนี้

attributes[0] = "01"
attributes[1] = "Luke"
attributes[2] = "Skywalker"
attributes[3] = "skywalker@rebelalliance.com"

ดังนั้นเราจะเอาข้อมูลนี้ใส่ลงในวัตถุนักศึกษา แล้วทำการเพิ่มนักศึกษาที่สร้างใหม่ลงไปใน students ที่เราสร้างไว้

Student studentFromCSV = new Student();
studentFromCSV.student_id = attributes[0];
studentFromCSV.firstname = attributes[1];
studentFromCSV.lastname = attributes[2];
studentFromCSV.email = attributes[3];
                
students.add(studentFromCSV);

นั่นคือข้อมูลนักศึกษาจาก csv จากการอ่านไฟล์ 1 บรรทัด แล้วนำไปสร้างวัตถุนักศึกษาแล้วทำการใส่ลงในข้อมูลนักศึกษาของคลาสรายวิชา

เราต้องทำซ้ำ อ่านข้อมูลไปทุก ๆ บรรทัด เพื่อให้ครบทุกบรรทัดในไฟล์ โดยทำการตรวจสอบเงื่อนไขในการอ่านแต่ละครั้งว่ามีข้อมูลนักศึกษาหรือไม่ โดยเช็คจากตัวแปร line ว่ามีข้อมูลหรือไม่

เพิ่มคำสั่งทำซ้ำ โดยตรวจสอบข้อมูลที่อ่านจากไฟล์ csv ในแต่ละบรรทัด

while (line != null) {

    Student studentFromCSV = new Student();

    studentFromCSV.firstname = attributes[1];
    studentFromCSV.lastname = attributes[2];
    studentFromCSV.email = attributes[3];
               
    students.add(studentFromCSV);
                
    line = br.readLine();
}

ปรับปรุงคำสั่ง เพิ่ม try catch เพื่อดังจับข้อผิดพลาดในกรณีที่โปรแกรมหาไฟล์จากตำแหน่งที่เรากำหนดไม่เจอ และเพิ่มตำแหน่งของไฟล์ที่เราจะอ่านให้สำมบูรณ์ เราจะได้เมธอท loadStudentFromCSV() สำหรับอ่านข้อมูลจากไฟล์ดังนี้

OOPClass.java

public void loadStudentFromCSV(){
        Path pathToFile = Paths.get("C:\\Users\\tanapattara\\Documents\\name.csv");
        try (BufferedReader br = Files.newBufferedReader(pathToFile, StandardCharsets.UTF_8)) {

            String line = br.readLine();

            while (line != null) {
                String[] attributes = line.split(",");

                Student studentFromCSV = new Student();
                studentFromCSV.student_id = attributes[0];
                studentFromCSV.firstname = attributes[1];
                studentFromCSV.lastname = attributes[2];
                studentFromCSV.email = attributes[3];
                
                students.add(studentFromCSV);
                
                line = br.readLine();
            }
        } catch (IOException ex) {
            ex.printStackTrace();
        }
    }

Warningr: ตรวจสอบตำแหน่งที่อยู่ของไฟล์ name.csv ในเครื่องให้ถูกต้อง โดยทำการตั้งที่อยู่ในคำสั่ง Paths.get("C:\\Users\\tanapattara\\Documents\\name.csv"); ให้ตรงกับเครื่องของตนเอง

ปรับคลาสรายวิชาให้ทำเรียกใช้งานเมธอท loadStudentFromCSV()เพื่อที่จะได้อ่านไฟล์ให้ csv ทันทีที่มีการสร้างคลาสรายวิชา

ทดสอบการทำงานของระบบ

เราได้สร้างคลาสรายวิชาที่เก็บข้อมูลนักศึกษาไว้ภายใน และลองเรียกดูข้อมูลอย่างง่าย หากเรามองในภาพรวมของระบบทั้งหมด สามารถแสดงเป็นแผนภาพคลาสดังภาพด้านล่าง

จากแผนภาพ เรามีคลาสรายวิชา ชื่อว่า OOPClass ภายในคลาสประกอบไปด้วยแฮททริคบิวท์ที่เก็บรายชื่อของนักศึกษาที่ลงทะเบียนเรียนเอาไว้ ในรูปแบบของ ArrayList มีเมธอท createStudent() เอาไว้สำหรับสร้างข้อมูลนักศึกษา เมธอท getStudentCount() เอาไว้แจ้งจำนวนนักศึกษาในชั้นเรียน และสุดท้ายเมธอท loadStudentFormCSV() สำหรับอ่านไฟล์ csv และสร้างวัตถุนักศึกษาจากข้อมูลที่อ่านได้ ยังมีคลาสนักศึกษา Student เอาไว้แทนรายละเอียดของนักศึกษา ประกอบไปด้วยแอททริบิวท์ ชื่อ นามสกุล รหัสนักศึกษา และอีเมล

เรายังได้ดึงข้อมูลนักศึกษาจากคลาสรายวิชา มาเก็บไว้ที่ตัวแปล students และแสดงข้อมูลนักศึกษาผ่านคำสั่งทำซ้ำ เพื่อแสดงรายข้อมูลนักศึกษาทุกคน
