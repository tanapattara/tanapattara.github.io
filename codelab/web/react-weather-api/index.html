ปรับปรุงล่าสุด: 2021-06-12

เราจะได้เรียนรู้การใช้งาน React ดึงข้อมูลผ่านเว็บบริการเพื่อนำข้อมูลที่ได้มาแสดงผลที่แอพพลิเคชั่นของเรา รวมถึงการใช้งานไลบรารี่อื่นๆ ที่มีให้ใน React Library

การติดตั้ง React

  1. ติดตั้ง Nodejs เวอร์ชั่นล่าสุดจากเว็บไซต์ nodejs.org
  2. ติดตั้ง react โดยใช้คำสั่ง npm i -g create-react-app จาก command line
  3. ติดตั้ง Visual Studio Code Editor
  4. ติดตั้ง Visual Studio Code Extension - ES7 React/Redux/GraphQL/React-Native snippets
  5. ติดตั้ง Visual Studio Code Extension - Prettier - Code Formatter (ตัวเลือกเพิ่มเติม)

Note: เราสามารถตั้งค่าให้ Visual Studio Code ทำการปรับปรุงโค้ด (Format) ให้อยู่ในรูปแบบอัตโนมัติหลังจากการบันทึกโดยไปที่เมนู Settings เลือก Editor: Format On Save

สิ่งที่จะได้จากบทเรียน

  • สร้างเว็บแอพพลิเคชั่น Weather อย่างง่ายโดยใช้ react ร่วมกับ

ความรู้พื้นฐานก่อนการทำเลป

  • HTML

  • Chrome DevTools.

ติดตั้ง library เพิ่มเติม

npm install semantic-ui-react semantic-ui-css
npm install moment --save

นำเข้า semantic css ที่ไฟล์ App.js

App.js

import 'semantic-ui-css/semantic.min.css'

เราจะทำการดึงข้อมูลมาจาก API ก่อนที่เราจะทำได้ต้องมีกุญแจสำหรับเข้าถึงบริการก่อน ในการทดลองนี้เราจะดึงข้อมูลสภาพอากาศ จากเว็บไซต์ OpenWeatherMap ให้เราไปที่เว็บไซด์ https://openweathermap.org/ ทำการสมัครสมาชิกให้เรียบร้อย

ที่เมนู API เราจะเห็นบริการที่เว็บไซด์นี้เปิดให้เราใช้งาน จะมีเอกสาร API Doc ให้เรา เราสามารถศึกษาข้อมูลการใช้งาน API ต่างๆ ได้จากเมนูนี้

สร้าง API Key ให้ไปที่เมนูชื่อเราและเลือก My API Key ให้เราสร้าง โดยการใส่ชื่อลงใน API Key name ทางขวามือ เป็นชื่อที่เราต้องการเรียกและกดปุ่ม Generate

เราจะได้ Key ทางขวามือ เพื่อนำไปใช้งานในโปรแกรมของเรา

ที่โฟลเดอร์หลักของเรา ให้สร้างไฟล์ที่ชื่อ .env โดยนำ API Key ที่เราสร้างมาใส่ลงไป พร้อมกับใส่การตั้งค่าต่อไปนี้

.env

REACT_APP_API_URL = 'https://api.openweathermap.org/data/2.5'
REACT_APP_API_KEY = Paste your API key here.
REACT_APP_ICON_URL = 'https://openweathermap.org/img/w'

นำเข้า useState และ useEffect จาก React Hook ในไฟล์ App.js

App.js

import React, { useEffect, useState } from "react";

สร้าง state สำหรับเก็บค่าตำแหน่งพิกัด

App.js

const [lat, setLat] = useState([]);
const [long, setLong] = useState([]);

สร้างฟังชั่นสำหรับดึงข้อมูลจากอุปกรณ์ เราจะเตรียมไว้สำหรับเรียกใช้งานเมื่อมีการเปิดแอพพลิเคชั่นขึ้นมาทำงาน

App.js

useEffect(() => {
    navigator.geolocation.getCurrentPosition(function(position) {
      setLat(position.coords.latitude);
      setLong(position.coords.longitude);
    });

    console.log("Latitude is:", lat)
    console.log("Longitude is:", long)
  }, [lat, long]);

ทดสอบการทำงานของแอพพลิเคชั่น โดยดูการแสดงผลผ่าน console โดยจะมีการขออนุญาติการเข้าถึงตำแหน่งจากผู้ใช้งานก่อน

ที่ OpenWeather เราจะเรียกใช้ API ทีชื่อ Current Weather Data โดยระบุตำแหน่งพิกัดที่เราได้จากผู้ใช้งานส่งไปยัง API ตามรายละเอียดที่ API กำหนดไว้ที่ Current weather data - OpenWeatherMap

API call

api.openweathermap.org/data/2.5/weather?lat={lat}&lon={lon}&appid={API key}

ในการเรียกใช้งาน api เราจะใช้ url ตามที่กำหนดมาให้ โดยเราจะเปลี่ยนแปลงพารามิตเตอร์ ตามค่าที่เราต้องการ นั่นก็คือค่าพิกัดที่เราต้องการสภาพอากาศ และ API Key ที่เราได้จากการสร้างในหน้าเว็บ

ทดสอบการเรียกใช้งาน โดยให้เราลง extension ที่ชื่อ REST Client เพื่อทำการทดสอบ

สร้างไฟล์ request.http ในโฟลเดอร์ source เพื่อทดสอบการทำงาน

API call

GET https://api.openweathermap.org/data/2.5/weather?lat=17.8262293&lon=102.7427118&appid={API key}

หลังจากนั้น คลิ๊กขวาที่ url และเลือก sent request จะได้รับข้อมูลจากเว็บบริการ API ตอบกลับมาในรูปแบบของ JSON ดังนี้

รูปทางขวาคือข้อมูลตอบกลับจากเว็บ API เป็นข้อมูลสภาพอากาศ ละรายละเอียดต่างๆเราจะนำข้อมูลนี้ไปแสดงผลที่เว็บไซด์ของเราอีกที

เมื่อทดสอบการเรียกใช้งานว่า เราสามารถเรียกใช้งานได้ถูกต้อง และได้รับข้อมูลจากเว็บแล้ว ให้เราเพิ่มโค้ดของเราในไฟล์ App.js ส่วนของการดึงข้อมูลพิกัดสถานที่ หลังจากได้แล้วก็ส่งข้อมูลไปที่เว็บ API โดยเรียกผ่านตัวแปลที่เราสร้างไว้ในไฟล์ .env

App.js

await fetch(`${process.env.REACT_APP_API_URL}/weather/?lat=${lat}&lon=${long}&units=metric&APPID=${process.env.REACT_APP_API_KEY}`)
      .then(res => res.json())
      .then(result => {
        setData(result)
        console.log(result);
      });

เราได้ใช้คำสั่ง fetch เรียกข้อมูลผ่าน url ที่เราได้กำหนดไว้ใน และได้มีการเพิ่มพารามิตเตอร์พิกัดและตำแหน่ง

เมื่อเราได้ข้อมูล คำสั่งจะทำการแปลงข้อมูลที่ได้จาก API เก็บไว้ในตัวแปร result และทำการบันทึกลงไปใน state ที่ชื่อ data

ที่ไฟล์ App.js เราจะได้ useEffect สำหรับการดึงข้อมูล API และการเก็บข้อมูลลง state ดังนี้

App.js

  const [lat, setLat] = useState([]);
  const [long, setLong] = useState([]);
  const [data, setData] = useState([]);

  useEffect(() => {
    const fetchData = async () => {
      navigator.geolocation.getCurrentPosition(function(position) {
        setLat(position.coords.latitude);
        setLong(position.coords.longitude);
      });

      await fetch(`${process.env.REACT_APP_API_URL}/weather/?lat=${lat}&lon=${long}&units=metric&APPID=${process.env.REACT_APP_API_KEY}`)
      .then(res => res.json())
      .then(result => {
        setData(result)
        console.log(result);
      });
    }
    fetchData();
  }, [lat,long])
  
  return (
    <div className="App">
      
    </div>
  );

ต่อไปเราจะเตรียมคอมโพแนนท์สำหรับแสดงผลสภาพอากาศจากข้อมูลที่เราได้รับมา ให้เราสร้างไฟล์ใหม่ชื่อ Weather.js โดยในคอมโพแนนท์นี้ใช้สำหรับแสดงผลอย่างเดียวเราสามารถสร้างเป็นแค่ฟังชั่นที่ใช้เก็บโครงสร้าง html สำหรับ render ก็เพียงพอ

สร้างไฟล์ Weather.js ที่โพลเดอร์ components และทำการนำเข้า sematic-ui-react เพื่อใช้งาน css ที่เราได้ทำการติดตั้งไว้ตั้งแต่ตอนแรก

Weather.js

import React from 'react'
import { Card } from 'semantic-ui-react'

const WeatherCard = ({weatherData}) => (
  <Card>
    <Card.Content>
         <Card.Header className="header">City Name: {weatherData.name}</Card.Header>
    </Card.Content>
  </Card>
)

export default WeatherCard

จะเห็นว่า เราสร้างตัวฟังก์ชั่นขึ้นมาในคอมโพแนนท์ แล้วทำการรับช้อมูลมาจากอาร์กูเม้นท์ที่ชื่อ weatherData และทำการนำไปแสดงผลชื่อเมืองโดยใช้คำสั่ง weatherData.name

ที่ App.js หลังจากที่ได้รับค่าช้อมูลจาก API และทำการบันทึกลง state แล้ว เราจะส่งค่าไปยังคอมโพเน้นท์ที่เราได้ทำการสร้างขึ้น โดยเราจะทำการเช็คก่อนว่า data.main มีข้อมูลจาก API แล้วหรือยัง

App.js

    <div className="App">
      {(typeof data.main != 'undefined') ? (
        <Weather weatherData={data}/>
      ): (
        <div></div>
      )}
    </div>

ทดสอบการทำงาน และดูการแสดงผลผ่านเว็บบราวเซอร์

จะเห็นได้ว่าข้อมูลพระอาทิตย์ขึ้น และตกนั้น เป็นข้อมูลตัวเลข ซึ่งไม่อยู่ในรูปแบบที่เราใช้งานทั่วไป ให้เราแปลงข้อมูลให้ตรงกับเวลาท้องถิ่งโดยใช้คำสั่ง toLocalTimeString

Weather.js

<p>Sunrise: {new Date(weatherData.sys.sunrise * 1000).toLocaleTimeString('en-IN')}</p>
<p>Sunset: {new Date(weatherData.sys.sunset * 1000).toLocaleTimeString('en-IN')}</p>

เรียกใช้ไลบรารี่ moment เพื่อใช้งานข้อมูลเวลาปัจจุบัน ผ่านคำสั่ง moment()

Weather.js

import moment from 'moment';
...
        <p>Day: {moment().format('dddd')}</p>
        <p>Date: {moment().format('LL')}</p>

ในกรณีที่เราทำการดึงข้อมูลจาก API บางครั้ง ยังไม่ได้ข้อมูลจาก API ทำให้ค่า data ยังเป็น undefined อยู่ ให้เราทำการตรวจสอบก่อนว่าค่าดังกล่าวได้รับข้อมูลมาจาก API หรือยัง หากว่ายังไม่มีข้อมูล ให้เรียกใช้ Loader ของ Semantic-ui

App.js

import { Dimmer, Loader } from "semantic-ui-react";

...

return (
    <div className="App">
      {typeof data.main != "undefined" ? (
        <Weather weatherData={data} />
      ) : (
        <div>
          <Dimmer active>
            <Loader>Loading..</Loader>
          </Dimmer>
        </div>
      )}
    </div>
  );

เนื่องจากว่าสภาพอากาศที่เราได้ เราสามารถใส่รูปภาพเพื่อเพิ่มความสวยงามให้กับการ์ดของเรา แต่ตอนนี้เราจะใส่ไอคอนให้กับการ์ดของเราก่อน โดยทำการเพิ่มไลบรารี่ที่ชื่อว่า react-fontawesome และ free-solid-svg-icons

terminal

npm install react-fontawesome
npm install free-solid-svg-icons

หลังจากนั้นให้เราทำการ importไลบราลี่เข้าไปยังคอมโพเน้นท์ที่เราต้องการใช้

Weather.js

import { FontAwesomeIcon } from "@fortawesome/react-fontawesome";
import {
  faBolt,
  faCloudShowersHeavy,
  faSun,
  faCloud,
  faSmog,
} from "@fortawesome/free-solid-svg-icons";

เราได้ทำการนำเข้าไอคอลเรียบร้อยแล้ว ต่อไปทำการตรวจสอบค่าที่ได้จาก API ว่ามีสภาพอากาศเป็นอย่างไร แล้วทำการเก็บไอคอลไว้ในตัวแปร

Weather.js

...
  let weatherIcon = null;

  if (weatherData.weather[0].main === "Thunderstorm") {
    weatherIcon = <FontAwesomeIcon icon={faBolt} />;
  } else if (weatherData.weather[0].main === "Rain") {
    weatherIcon = <FontAwesomeIcon icon={faCloudShowersHeavy} />;
  } else if (weatherData.weather[0].main === "Clear") {
    weatherIcon = <FontAwesomeIcon icon={faSun} />;
  } else if (weatherData.weather[0].main === "Clouds") {
    weatherIcon = <FontAwesomeIcon icon={faCloud} />;
  } else {
    weatherIcon = <FontAwesomeIcon icon={faSmog} />;
  }
...

หลังจากนั้น นำตัวแปรที่ได้ ไปใส่ในการ์ดในตำแหน่งที่เราต้องการ
