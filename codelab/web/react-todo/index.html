ปรับปรุงล่าสุด: 2021-06-12

การทดลองนี้จะเป็นการสร้างแอพพลิเคชั่น Todo สำหรับบันทึกรายการที่ต้องทำลงในแอพ โดยกรอกรายละเอียดลงไปในกล่องรับข้อความเมื่อมีการกด add จะแสดงรายการด้านล่าง และยังสามารถกดปุ่ม ลบ และแก้ไขรายการได้อีกด้วย

ด้วยความสามารถของ React ที่แยกการทำงานของหน้าจอออกเป็นคอมโพแนนท์ย่อย เราจะมาดูรายละเอียดการส่งข้อมูล และฟังก์ชั่นให้กับคอมโพแนนท์อื่น รวมถึงภาพรวมของโปรแกรม และการตกแต่งด้วย css

การติดตั้ง React

  1. ติดตั้ง Nodejs เวอร์ชั่นล่าสุดจากเว็บไซต์ nodejs.org
  2. ติดตั้ง react โดยใช้คำสั่ง npm i -g create-react-app จาก command line
  3. ติดตั้ง Visual Studio Code Editor
  4. ติดตั้ง Visual Studio Code Extension - ES7 React/Redux/GraphQL/React-Native snippets
  5. ติดตั้ง Visual Studio Code Extension - Prettier - Code Formatter (ตัวเลือกเพิ่มเติม)

Note: เราสามารถตั้งค่าให้ Visual Studio Code ทำการปรับปรุงโค้ด (Format) ให้อยู่ในรูปแบบอัตโนมัติหลังจากการบันทึกโดยไปที่เมนู Settings เลือก Editor: Format On Save

สร้างฟอร์มสำหรับรับข้อมูลจากผู้ใช้

สร้างฟอร์มสำหรับให้ผู้ใช้งานกรอกข้อมูลเข้าไปยังโปรแกรม เราใช้ input และ button ปุ่มสำหรับรับข้อมูล ที่ไฟล์ TodoForm.js เราจะได้คอมโพแนนท์ตามคำสั่งด้านล่าง

TodoForm.js

import React, { useState } from "react";

function TodoForm() {
  const [input, setInput] = useState("");
  return (
    <form className="todo-form">
      <input
        type="text"
        placeholder="Add a todo"
        value={input}
        name="text"
        className="todo-input"
      />
      <button className="todo-button">Add todo</button>
    </form>
  );
}

export default TodoForm;

ในตอนนี้เราจะมีคอมโพแนนท์อยู่สองตัว ที่หน้าหลักจะเป็น TodoList คอมโพแนนท์ ภายในมี TodoForm คอมโพแนนท์สำหรับรับข้อมูลจากผู้ใช้งาน

ตอนนี้เรามีคอมโพแนนท์สามอัน แยกเป็น 1) คอมโพแนนท์สำหรับเก็บข้อความจากผู้ใช้งาน TodoForm.js 2) คอมโพแนนต์สำหรับแสดงผลข้อความจากผู้ใช้ Todo.js และสุดท้าย 3) คอมโพแนนท์ที่รวมสองคอมโพแนนท์แรกเข้าด้วยกัน TodoList.js ทำให้สามารถเขียนรูปได้ตามภาพ

เราจะทำการนำข้อมูลจาก TodoForm มาแสดงที่ Todo ดังนั้น เราจำสร้างตัวแปรและฟังชั่นที่ TodoList ที่ใช้สำหรับเก็บข้อมูลและส่งไปให้คอมโพแนนต์ย่อยทั้ง TodoForm และ Todo ใช้งาน

ที่ไฟล์ TodoList.js เราจะสร้างฟังก์ชั่นสำหรับดึงข้อมูลจาก input แล้วทำการเซ็ตค่าที่ได้ลง state ของคอมโพแนนท์ TodoList ดังนั้น เราจะส่งฟังก์ชั่นดังกล่าวไปเป็นพร๊อพเพอร์ตี้ ของ TodoForm เพื่อให้คอมโพแนนท์สามารถส่งค่าขึ้นมาที่ List ได้

สร้างตัวแปรใน state และฟังชั่น

ที่ TodoList สร้าง state ชื่อ todos เพื่อใช้เก็บค่าที่ได้รับจากผู้ใช้งาน โดยให้กำหนดค่าเริ่มต้นเป็นแอร์เรย์เปล่า และสร้างฟังชั่น addTodo สำหรับส่งและให้ TodoForm เรียกใช้

TodoList.js

const [todos, setTodos] = useState([]);

const addTodo = (todo) => {
    const newTodos = [todo, ...todos];

    setTodos(newTodos);
    console.log(todo);
  };

จะเห็นว่า addTodo จะทำการรับค่า Todo เข้ามาและทำการใส่ลงไปแอร์เรย์ todos โดยใช้คำสั่ง [todo, ...todos] หลังจากนั้นทำการบันทึกลง todos ที่เป็น State ของคอมโพแนนต์ และทำการแสดงผลออกทาง log เพิ่มตรวจสอบการทำงาน

ส่งฟังก์ชั่นไปยังคอมโพเนนท์ TodoForm ผ่านพร๊อพเพอร์ตี้ของ TodoForm

คำสั่งด้านบนจะตรวจสอบค่า หากมีค่าเปลี่ยนแปลงจาก null นั่นคือผู้ใช้มีการกดปุ่ม edit ทำให้ state มีการเปลี่ยนแปลงค่าจากเดิม จะทำการส่งคอมโพแนนท์ TodoForm กลับ พร้อมกับเรียกใช้ฟังชั่น submitUpdate เมื่อกดยืนยัน และอย่าลืมเพิ่ม import TodoForm ในส่งหัวของไฟล์เพิ่มเรียกใช้งานคอมโพแนนท์ให้ถูกต้อง
