JavaScript คืออะไร

JavaScript หรือบางคนเรียกสั้นๆว่า JS ได้ถูกพัฒนาเพื่อให้ทำงานบนเว็บบราเซอร์ เรียกว่า JavaScript engine บนบราวเซอร์ FireFox เรียกว่า SpiderMonkey และบนบราวเซอร์ Chrome เรียกว่า v8 เราสามารถเรียกใช้งาน JS ผ่านเว็บบราวเซอร์ หรือสามารถใช้งานผ่าน Node.JS ที่ทาง google ได้ทำการนำ Google V8 engine ออกมาจากบราวเซอร์ ทำให้จาวาสคริปต์สามารถทไงานได้ทั้งบนเว็บไซต์ เว็บบริการ เว็บแม่ข่าย หรือแม้กระทั้งบนมือถือ

เรากำลังจะทำอะไร

ในตัวอย่างนี้เราจะได้เรียนรู้การใช้ JS บนเว็บบราวเซอร์ การประกาศตัวแปรต่างๆ พื้นฐานการใช้งาน JS

เราจะได้เรียนรู้อะไร

  • วิธีการสร้างตัวแปร
  • วิธีการสร้างวัตถุ
  • วิธีการสร้างฟังชั่น

เตรียมตัวก่อนการทดลอง

  • เว็บบราวเซอร์ ในการทดลองใช้ตัวอย่างจาก Google Chrome สามารถใช้เว็บบราวเซอร์ตัวอื่น ๆ ได้ตามความถนัด ขอให้เป็นรุ่นล่าสุด
  • ตวามรู้พื้นฐานของ HTML, CSS, JavaScript และ Chrome DevTools.
  • Setting Editor
  • Visual Studio Code Editor
  • Live Server Extension ส่วนเสริมบน Visual Studio Code Editor
  • Nodejs.org (Optional)

ทดสอบการทำงานบนบราวเซอร์

เปิด Google Chrome DevTools โดยการกดปุ่ม F12 หลังจากนั้นทำการพิมพ์คำสั่งด้านล่างที่หน้าต่าง Console panel.

console.log("Hello World");

กดปุ่ม Enter เพื่อดูผลลัพท์การทำงาน

เราสามารถใช้งาน Expression บนหน้าต่างคอลโซลได้ โดยการพิมพ์การคำนวนแล้วกด enter เพื่อดูผลการทำงาน

2 + 2

ตัวอย่างอื่น ๆ เช่นการเรียกใช้งานกล่องแจ้งเตือน โดยใช้คำสั่ง `alert` ตามตัวอย่างข้างล่าง

alert(‘Hello JS')

Tip: จะเห็นได้ว่า เราไม่จำเป็นต้องปิดคำสั่งโดยใช้เครื่องหมายเครื่องหมายอัฒภาค หรือเครื่องหมาย semicolon (;) ในการทำงานของสคริปต์ และเราสามารถใช่เครื่องหมายฝนทอง (single quoted) และเครื่องหมายฟันหนู (double quoted) ได้ในการใช้งานการแสดงผลอักขระ

การใส่แบบ Inline script

สร้างไฟล์ index.html และใส่โค้ด html5 อย่างง่าย

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JavaScript Lab</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

บราวเซอร์อ่านไฟล์จากบนลงล่าง ถ้าเราทำการเพิ่มคำสั่งสคริปต์ลงในส่วนบนของหน้าเว็บ บราวเซอร์จะทำการอ่านชุดคำสั่งก่อนการอ่านข้อมูลหน้าเว็บซึ่งเป็นส่วนการแสดงผล ส่วนติดต่อผู้ใช้ ทำให้เว็บไซด์จะปรากฎต่อผู้ใช้งานได้ช้า เนื่องจากบราวเซอร์จะทำการอ่านสคริปต์ก่อน ดังนั้นเรานิยมนำคำสั่งสคริปต์ไว้ล่าสุด ในสวนของแท็ก

ทำการเพิ่มสคริปต์โดยใช้แท็ก

index.html

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JavaScript Lab</title>
</head>
<body>
    <h1>Hello World</h1>
    <script>
     //this is a comment
     console.log(‘Hello world');
    </script>
</body>
</html>

ทำการเปิด Liver server และดูผลลัพท์การทำงานผ่าน ChromeDev tools console

การใส่แบบแยกไฟล์

HTML ควรเป็นไฟล์ที่เก็บข้อมูลส่วนของการแสดงผลของหน้าเว็บเป็นหลัก ดังนั้นเราควรแยกคำสั่ง JS ออกจากไฟล์ HTML เพื่อง่ายต่อการจัดการและการพัฒนา

สร้างไฟล์ใหม่ ชื่อ index.js ไว้ในโฟลเดอร์เดียวกันกับไฟล์ index.html โดยในไฟลของ JS ให้ใช้คำสั่งดังนี้

index.js

console.log(‘Hello world');

ที่ไฟล์ index.html ให้ทำการแห้ไขส่วนของการเรียกใช้งาน โดยแก้ไขคำสั่ง ในการเขียนคำสั่ง JS

ทดสอบและดูผลการทำงาน

การสร้างตัวแปร Variable

ที่ไฟล์ index.js ทำการสร้างตัวแปรชื่อ name โดยใช้คำสั่ง let แล้วตามตัวชื่อตัวแปร

index.js

let name;
console.log(name);

ทดสอบการทำงาน แสดงผลการทำงานผ่าน log ผลลัพท์ที่ได้เราจะได้ค่า undefined นั่นคือค่าปริยายของตัวแปรใน JS

ต่อไปทำการใส่ค่าลงไปในตัวแปรที่ได้สร้างขึ้น ใช้เครื่องหมาย = ในการกำหนดค่าให้กับตัวแปร

index.js

let name = ‘JavaScript';
console.log(name);

ผลการทำงานจะเป็นการแสดงค่า ‘JavaScript' ออกทางหน้าต่าง Console

สำหรับตัวแปรอื่น ๆ ใช้กฎพื้นฐานในการตั้งชื่อตัวแปร เช่นเดียวกับการตั้งชื่อตัวแปรในภาษาคอมพิวเตอร์ภาษาอื่นๆ

  • ไม่สามารถสร้างตัวแปรเป็นชื่อเดียวกับคำสงวน (reserved keyword) ในภาษานั้น ๆ
  • ต้องมีความหมายต่อค่า และการใช้งาน
  • ไม่สามารถขึ้นต้นด้วยตัวเลขได้
  • ไม่สามารถเว้นช่องว่าง (space) [ ] หรือมีเครื่องมายยัติภังค์ (hyphen) [-]
  • การสร้างชื่อตัวแปรเป็นลักษณะ case-sensitive (firstName, FirstName, Firstname)

ค่าคงที่ Constants

Create variable name interestRate at index.js

ที่ไฟล์ index.js ทำการสร้างตัวแปรที่ชื่อ interestRate และทำการกำหนดค่าให้กับตัวแปร

หลังจากนั้นทำการกำหนดค่าตัวแปร interestRate ใหม่ โดยเปลี่ยนค่าให้เป็น 1

index.js

let interestRate = 0.3;
interestRate = 1;
console.log(interestRate);

ทดสอบการทำงานบนบราวเซอร์ เราจะพบว่าค่าของตัวแปร interestRate ได้มีการเปลี่ยนแปลงจากเดิมเป็นค่าใหม่ ในบางกรณีเราไม่ต้องการให้ค่าของตัวแปรที่เราสร้างขึ้นมามีการเปลี่ยนแปลงไปจากเดิม เราสามารถกำหนดให้ตัวแปรนั้นเป็นค่าคงที่ โดยทำการสร้างตัวแปรแบบ constants โดยการใช้คีย์เวริส์ const

เปลี่ยคำสั่งก่อนหน้า จาก let เป็น const

index.js

const interestRate = 0.3;
interestRate = 1;
console.log(interestRate);

ทดสอบการทำงานที่บราวเซอร์ เราจะเห็นการแจ้งเตือนความผิดพลาด Uncaught TypeError: Assignment to constant variable. นั้นคือเราไม่สามารถเปลี่ยนแปลงค่าตัวแปรที่เป็นชนิด const ได้หลักจากการประกาศ

ชนิดของตัวแปร Primitives type

มีค่าอะไรบ้างที่เราสามารถใส่ลงไปให้กับตัวแปรของเราได้ ชนิดของตัวแปรหลัก ๆ ใน JS ประกอบไปด้วย

  • String
  • Number Boolean
  • Undefined
  • Null

ตัวอย่างชนิดของค่าที่สามารถเก็บลงตัวแปรได้ใน JS

index.js

let name = "Tanapattara" ; // String Literal
let class = 2020; //Number Literal
let isRegisted = false; // Boolean Literal
let firstName = undefined;
let lastName = null; // we used this to clear a value of variable

ตัวแปรใน JS แตกต่างจากภาษาโปรแกรมอื่นๆ ที่สามารถเปลี่ยนแปลงชนิดของตัวแปรได้ เราเรียกว่า Dynamic type ในภาษาโปรแกรมอื่นเราต้องกำหนดชนิดของตัวแปร ในขั้นตอนของการประกาศตัวแปรและไม่สามารถเปลี่ยนได้ แต่ใน JS สามารถเปลี่ยนได้ตลอดเวลา ในขณะทำงาน (Runtime) เราเรียกว่า dynamically-typed.

ที่ตัวแปรเดิมที่ได้มีการประกาศไว้

index.js

let name = "Tanapattara" ; // String Literal
let class = 2020; //Number Literal
let isRegisted = false; // Boolean Literal
let firstName = undefined;
let lastName = null; // we used this to clear a value of variable

ทดสองเปลี่ยนชนิดข้อมูลโดยการกำหนดค่าใหม่ให้กับตัวแปรโดยตรง

index.js

name = 2020;

ทำการแสดงผลบนบราวเซอร์ โดยใช้คำสั่ง typeof name เราจะเห็นผลลัพท์ชนิดของตัวแปรได้เปลี่ยนเป็น number จากที่เราได้ใส่ค่าชนิด String ไว้ตั้งแต่ตอนสร้างครั้งแรก.

การสร้างตัวแปรเราสามารถใช้คีย์เวิร์ด var, let หรือ const ได้ แล้วทั้งสามต่างกันอย่างไรในการสร้างตัวแปร

มาทดสองสร้างตัวแปรขึ้นมาโดยใช้คีย์เวิร์ดทั้งสามดังคำสั่งด้านล่าง

if(true){
    var varVariable = true;
}

console.log(varVariable);

if(true){
    let letVariable = true;
    const constVariable = true;
}

console.log(letVariable);
console.log(constVariable);

ทดสอบการแสดงผลบนบราวเซอร์เราจะเห็นได้ว่า varVariable เป็นเพียงตัวแปรเดียวที่แสดงผลในคำสั่ง log หากเรามองดี ๆ ตัวแปรที่สร้างจากคำสั่ง var ได้ทำการประกาศไว้ในวงเล็บของ if แต่สามารถแสดงผลนอกคำสั่ง if ได้นั่นก็เพราะ var เป็น function scrope สามารถเข้าถึงได้ทุกที่ ในขณะที่ สามารถเข้าถึงได้เฉพาะในบล๊อคของที่มันถูกสร้างขึ้นเท่านั้น

ทดลองเปลี่ยนการประกาศตัวแปรให้อยู่ในของเขตเดียวกัน

var varVariable = true;
var varVariable = false;

console.log(varVariable);

let letVariable = true;
let letVariable = false;

const constVariable = true;
const constVariable = false;

console.log(letVariable);
console.log(constVariable);

ทดสอบการทำงานบนบราวเซอร์ เราจะเป็นได้ว่า มีข้อผิดพลาดแสดงว่า เราไม่สามารถประกาศตัวแปรซ้ำกันชื่อเดียวกันในคำสั่ง let และคำสั่ง const ในทางตรงกันข้าว เราสามารถประกาศตัวแปรที่มีชื่อเดียวกันโดยใช้ตัวแปร var ได้โดยไม่มีข้อผิดพลาด ในกรณีนี้ var สามารถอาจจะทำให้เกิดข้อผิดพลาดในการสร้างหรือประกาศตัวแปรใหม่ได้ในกรณีที่ใช้ชื่อเดียวกัน

ฟังก์ชั่นคือหลักพื้นฐานอขงการสร้าง blocks ใน JS มาลองสร้างฟังก์ชั่นสักอัน

function greet(){
   //logic of this function
   console.log('Hello world');
}

Note: เราไม่จำเป็นต้องใส่ semicolons ปิดท้ายในคำสั่งฟังก์ชั่น

เมื่อสร้างแล้ว เราสามารถเรียกใช้ฟังก์ชั่นโดยการระบุชื่อฟังก์ชั่นได้เลย

function greet(){
   //logic of this function
   console.log('Hello world');
}

// call a function
greet();

ฟังก์ชั่นนั้น เราสามารถใส่ข้อมูลให้กับฟังชั่นได้ โดยการกำหนดพารามิตเตอร์ให้กับฟังชั่น โดยใส่เป็นชื่อตัวแปรได้เลย ทดสอบโดยกำหนดให้รับข้อมูลเป็นข้อความแล้วมาแสดงผลออกทาง console โดยเราจะได้ฟังก์ชั่นดังนี้

function greet(name){
   //logic of this function
   console.log('Hello ' + name);
}

// call a function
greet('John');

เราส่งค่า John เป็นอาร์กูเม็นท์ argument ผ่านฟังก์ชั่น greet โดยฟังก์ชั่นจะนำค่าที่ได้รับมาใส่ในตัวแปร name ในรูปแบบของพารามิเตอร์ parameter

เราสามารถเพิ่มพารามิเตอร์เข้าไปให้กับฟังก์ชั่นเท่าไหร่ก็ได้ โดยการเพิ่มตัวแปรเข้าไปในวงเล็บของฟังก์ชั่น หากเราเรียกใช้ฟังก์ชั่น เราต้องส่งค่าผ่านอาร์กูเม็นท์ที่เท่ากันกับฟังชั่นต้องการ ดังตัวอย่ามีการเพิ่มพารามิเตอร์ให้กับฟังก์ชั่น greet คือ lastname ดังนั้นหากเราต้องการเรียกใช้ฟังก์ชั่น เราก็ต้องส่งการ์กูเม็นท์มาให้ตรงกับที่ต้องการ

function greet(name, lastname){
   //logic of this function
   console.log('Hello ' + name + ' ' + lastname);
}

// call a function
greet('John', 'Doe');

Caution: ตัวแปรที่เป็นพารามิเตอร์สามารถใช้ได้เพียงในฟังก์ชั่นที่ถูกสร้างขึ้นเท่านั้น ไม่สามารถเรียกใช้นอกฟังก์ชั่นได้ นั่นก็คือตัวแปร name และ lastname ไม่สามารถเรียกใช้งานได้นอกฟังชั่น greet

การส่งค่ากลับ

เราสามารกำหนดให้ฟังชั่นที่เราสร้างขึ้น สามารถส่งค่ากลับไปโดยใช้คำสั่ง return ในฟังชั่น ตัวอย่างเช่นการส่งค่าอุณหภูมิองศาเซลเซียส ไปแปลงเป็นองศาฟาเรนไฮน

function celsius2fahrenheit(celsius){
   Return celsius * (9 / 5) + 32;   
}

// call a function
let f = celsius2fahrenheit(100);

อย่างที่ทราบกัน วัตถุ (Object) คือแนวความคิดในโลกความจริงในการนำหลักการสร้างวัตถุรวมกันกับตัวแปร ฟังชั่น และโครงสร้าง

ตัวอย่างเช่น การจะแทนบุคคลในโลคความเป็นจริงเข้าไปในภาษาโปรแกรม โดยแทนวัตถุนั้นเป็นบุคคล โดยมีคุณสมบัติ (properties) ของวัตถุเช่น ชื่อ อายุ ส่วนสูง น้ำหนัก และมีการกระทำระหว่างวัตถุด้วยกันเช่น เดิน วิ่ง เปิดประตู เราสามารถสร้างวัตถุของบุคคลใน JS ได้ดังนี้

index.js

let person = {
    name: 'Tanapattara',
    age: 29
 };

ทดลองแสดงค่าวัตถุ person ผ่านคำสั่ง console.log(person)

หากเราต้องการเปลี่ยนค่าในวัตถุ เราสามารถใช้จุด dot notation หรือกล่อง bracket notation เพื่อทำการอ้างถึงแอททริบิวต์ของวัตถุได้โดยตรง ตัวอย่างเช่น การแก้ไขชื่อของวัตถุ person

index.js

let person = {
    name: 'Tanapattara',
    Age: 29
 };

// Dot Notation
person.name = 'John';
// Bracket Notation
Person['name'] = 'John Doe';

console.log(person.name);

หากเราต้องการสร้างวัตถุ เพื่อนำไปใช้สร้างวัตถุอื่นๆ เช่นเดียวกับการสร้างคลาสในภาษาอื่นๆ เราสามารถใช้ฟังชั่นในการสร้างโดยมีคำสั่งดังนี้

function Person(name, age, sex) {
  this.name = name;
  this.age = age;
  this.sex = sex;
}

เราจะสร้างวัตถุจากฟังชั่น Person และใส่ข้อมูลพื้นฐานให้กับวัตถุ โดยใช้คำสั่ง 

var rand = new Person('Rand McKinnon', 33, 'M');
var ken = new Person('Ken Jones', 39, 'M');

เราจะได้วัตถุ Person สองตัวโดยมีคุณสมบัติของวัตถุที่เหมือนกันตามที่เราได้ออกแบบไว้ในฟังชั่น Person

สร้างวัตถุซ้อนวัตถุ

เมื่อเราได้วัตถุ Person แล้วเราสามารถใส่วัตถุลงไปใส่วัตถุได้โดยการส่งวัตถุไปในขั้นตอนการสร้าง ตัวอย่างเช่น สร้างวัตถุ Car โดยให้มีแอททริบิวต์เจ้าของรถยนต์ด้วย โดยแอททริบิวต์เจ้าของรถยนต์นั้น เป็นวัตถุ Person

function Car(make, model, year, owner) {
  this.make = make;
  this.model = model;
  this.year = year;
  this.owner = owner;
}

ทำการสร้างวัตถุ Car โดยใช้คำสั่งและส่งพารามิตเตอร์ไปกับฟังชั่นตามคำสั่งข้างล่าง

var car1 = new Car('Eagle', 'Talon TSi', 1993, rand);
var car2 = new Car('Nissan', '300ZX', 1992, ken);

ทำการเรียกข้อมูลที่ที่อยู่ภายใน โดยใช้เครื่องหมายจุด เพื่อเรียกข้อมูล ตัวอย่างเช่น

car2.owner.name

การเขียนโปรแกรม JS โดยใช้หลักการ OOP จะอยู่ในอีกการทดลองหนึ่งซึ่งละมีคุณสมบัติของ OOP ครบถ้วนกว่า

อาร์เรย์เป็นการเก็บข้อมูลหลาย ๆ ตัวไว้ในตัวแปรเดียว ใน JS สามารถเก็บข้อมูลในรูปแบบอาร์เรย์ได้ทั้งในรูปแบบชนิดเดียวกัน และต่างชนิดได้

ทดลองสร้างตัวแปรที่เก็บข้อมูลสื

index.js

let colors = ['red', 'blue', 'green'];
console.log(colors);

การแสดงผลเราจะเห็นข้อมูลทั้งหมดของตัวแปร colors ที่เก็บข้อมูลไว้ โดนเรียงลำดับข้อมูลเริ่มจาก 0 เราสามารถเข้าถึงแต่ละค่าในอาร์เรย์ได้โดยใส่ตัวชี้ index ให้กับตัวแปร เช่น colors[0]

อย่างที่บอกไป JS เป็น dynamic language นั่นหมายถึงชนิดของตัวแปรสามารถเปลี่ยนแปลงได้ขณะทำงาน (runtime)

ทดลองเพิ่มข้อมูลอาร์เรย์ในตำแหน่งที่ 3 เราจะใช้คำสั่งตัวแปรตามด้วยตำแหน่ง และกำหนดค่าให้กับตำแหน่งนั่นได้ทันที

index.js

let colors = ['red', 'blue', 'green'];
Colors[3] = 'pink';
console.log(colors);

ผลลัพท์ที่ได้ colors จะมีข้อมูลเพิ่มมาในตำแหน่งที่ 3

ทดลองใส่ข้อมูลอื่นที่เป็นชนิดของตัวแปรแตกต่างกัน โดยกำหนดค่าลงไปที่ตำแหน่งที่ 4

let colors = ['red', 'blue', 'green'];
Colors[3] = 'pink';
Colors[4] = 1;
console.log(colors);

อาร์เรย์เป็นตัวแปรชนิดวัตถุ ทำให้มีคุณสมบัติที่ JS เตรียมไว้ให้และได้รับการสืบทอดมาตามคุณสมบัติของ OOP ให้เราได้ใช้งาน เช่นคำสั่ง colors.length.

แสดงผลข้อมูลโดยการทำซ้ำ

เนื่องจากเราสามารถเข้าถึงข้อมูลของอาร์เรย์โยใช้ตัวชี้อ้างอิงตำแหน่ง ดังนั้นเราสามารถใช้คำสั่งทำซ้ำในการดึงข้อมูลทุกตัวออกมาได้ เช่น

let color = ["red", "blue", "green"];      
for (let i = 0; i < color.length; i++) {
   console.log(color[i]);
}

หรือการใช้ FOR OF ในการเข้าถึงอาร์เรย์ทุกตัวโดยไม่อ้างอิงตำแหน่ง

for (let c of color) {
  console.log(c);
}

ทดลองสร้างฟังชั่นโดนใข้ JS โดยกำหนดให้สร้างฟังก์ชั่นแฟกทอเรียล และทดสอบการทำงาน

index.js

function factorial(number){
    //logic of this function
    return number;
}
 
let factorialNumber = 5;
 // call a function
console.log(factorial(factorialNumber));

ให้เพิ่มเติมฟังก์ชั่นให้สมบูรณ์และทดสองการทำงาน

ท่านได้เรียนรู้พื้นฐานการใช้งาน JS ผ่านทางเว็บบราวเซอร์ การประกาศตัวแปร การสร้างฟังก์ชั่น การสร้างวัตถุ และการเรียกใช้งานขั้นพื้นฐานเกี่ยวกับ JS

ต่อไปจะเป็นการนำ JS ไปใช้ร่วมกับ HTML ในการเข้าถึง DOM เพื่อทำงานบนเว็บเพจ
