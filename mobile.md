# 935 304 WEB APPLICATION PROGRAMMING

การเขียนโปรแกรมประยุกต์บนเว็บ

### คำอธิบายรายวิชา

สถาปัตยกรรมฮาร์ดแวร์ คุณลักษณะและข้อจํากัดของอุปกรณ์เคลื่อนที่ เครื่องมือและภาษาที่ใช้สําหรับพัฒนาโปรแกรมประยุกต์บนอุปกรณ์เคลื่อนที่ กระบวนการพัฒนาโปรแกรมประยุกต์บนอุปกรณ์เคลื่อนที่ การใช้หน่วยความจําและส่วนเก็บบันทึกข้อมูล การติดต่อกับผู้ใช้ การสื่อสารเครือข่ายกับภายนอก การเชื่อมโยงกับระบบคอมพิวเตอร์ การทดสอบโปรแกรมประยุกต์บนอุปกรณ์เคลื่อนที่โดยใช้ระบบคอมพิวเตอร์ ประเด็นด้านความมั่นคง การฝึกปฏิบัติ
